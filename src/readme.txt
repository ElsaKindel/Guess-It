Projet Guess It

Le but du jeu est de trouver le mot dont les définitions sont affichées.
Pour commencer cliquez sur le bouton "Commencer le jeu".
Ecrivez ensuite le mot dans le champ de réponse en haut à gauche, si rien ne se produit c'est que vous n'avez pas trouvé la bonne réponse.
Pour afficher la réponse, cliquez sur le bouton "Réponse".
Pour avoir les définitions d'un nouveau mot, cliquez sur "Jouer à nouveau".
